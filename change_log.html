Change List and Justification

From Dust to Detail: Deep Learning Pipelines for Analog Film Restoration

Author: Pol Riubrogent Comas

Initial Objectives and Tasks

The project was initially conceived as a research and development effort aimed at restoring degraded analog film using deep learning techniques. Four main objectives were outlined:

  • O1: Propose a restoration solution for scanned 35mm and 16mm film reels from the 1960s, focusing on removing dirt and scratches while preserving the characteristic film grain.
  • O2: Obtain or build a usable dataset that accurately represents common film degradations such as dust, dirt, and scratches.
  • O3: Develop a visually appealing and accessible graphical user interface to present results in a user-friendly way.
  • O4: Extend the system’s capabilities to generalize to other types of film stock beyond the initial target.

The technical tasks defined to fulfill these objectives were:

  • T1: Research existing datasets for film restoration.
  • T2: Generate a synthetic dataset using high-resolution clean frames and artifact masks.
  • T3: Implement a segmentation model to detect defects in individual frames.
  • T4: Integrate temporal context from adjacent frames to improve segmentation.
  • T5: Implement a basic inpainting pipeline to test restoration from segmented masks.
  • T6: Explore alternative inpainting architectures for comparative evaluation.
  • T7: Develop a graphical user interface to showcase the full pipeline.

Changes and Justifications

Initial Report → Progress Report I

  • Real datasets were unavailable, so a synthetic dataset was created using the FILM-AA framework.
  • Segmentation architecture was fixed as U-Net and its variants; transformer-based models were discarded due to complexity.
  • RePaint was adopted for inpainting, replacing the idea of training a model from scratch.

Dropped Task:

  • T4: Temporal modeling was not implemented due to time constraints and its architectural complexity. It will be pursued independently in the future.

Progress Report I → Progress Report II

  • The dataset was improved with Gaussian blur on artifact edges to mimic real damage.
  • Pretraining with the Documartica dataset improved generalization.
  • Evaluation was formalized using IoU, Dice, PSNR, SSIM, and LPIPS metrics.

Dropped Task:

  • T6: Comparison of multiple inpainting architectures was skipped. RePaint was sufficient and modular; effort was redirected to evaluation and analysis.

Progress Report II → Final Report

  • Introduced a second pipeline based on Restormer, offering an end-to-end transformer-based restoration strategy.
  • Conducted full quantitative and qualitative evaluations, including real archival footage.

Dropped Items:

  • T7: The GUI was dropped. As the project shifted toward a comparative study of restoration models rather than a deliverable product, interface development was no longer a priority.
  • O4: Generalization to other film types was not achieved. The project remained focused on the 1960s film domain for reliability and scope management.

Final Remarks

The project fulfilled its main research goal: restoring analog film while preserving its aesthetic integrity. The development culminated in two complete approaches—one modular and one end-to-end—and offered a technical comparison backed by metrics and real-world testing.

While the GUI and temporal modeling were not implemented, these decisions reflect a focus on scientific contribution over feature breadth. Both elements are planned for future independent development.
